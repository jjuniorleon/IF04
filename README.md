# Sobre a atividade

 
 O programa em C solicita que o usuário escolha um número, lê o número inserido e verifica se é positivo. Se for positivo, calcula e exibe a raiz quadrada e o quadrado desse número com duas casas decimais. Caso contrário, exibe a mensagem "escolha um número positivo". Após essa verificação e possível cálculo, o programa encerra sua execução.
